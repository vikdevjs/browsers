# Browser installer for windows
 
 To run this Follow these Steps
 
 ## installation
   ```
   git clone https://github.com/vikdevjs/browsers.git
   ```
 ## usage
  ``` 
  cd browsers
  ```
  ```
  .\browserInstaller.bat
  ```
  
  After this the Script will will run && Follow Steps in script.
